


## File format reference:
 \# - comment  
 @ - new set  
 $ - new section  
 ! - entry with mistake  
 ^ - mastered entry  
answer modifiers:  
 (no symbol) - mandatory answer  
 - - plural / ending
 | - alternative to previous answer  
 \* - non-mandatory answer  
 
Special symbols work only if they are the first symbol in the line / word (not counting whitespaces)
